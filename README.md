# openmemory-vercel-ai-starter

starter for using openmemory with vercel ai sdk.

## setup

1. install dependencies:
   ```bash
   npm install
   ```

2. set `OPENAI_API_KEY` in `.env` (or environment).

3. run:
   ```bash
   npm start
   ```

## what it does

uses `generateText` with openmemory as a tool.
